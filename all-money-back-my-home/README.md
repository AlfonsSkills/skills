# all-money-back-my-home

## 简介
这是一个用于分析投研 PDF 报告的通用 skill，不限于 Codex 使用场景。它能从报告中提取股票信息、识别板块与推荐等级，并生成结构化的投资策略表格，帮助用户快速理解报告的核心结论。

## 主要功能
- 提取报告中提到的股票名称
- 识别每只股票的所属板块（基于上下文判断）
- 根据报告措辞判断推荐等级
- 提取核心推荐理由
- 补充对应的股票代码（A股/港股/美股）
- 输出结构化 Markdown 表格

## 工作流
1. 解析 PDF 内容（优先使用 `pdftotext`，备选 `PyPDF2`）
2. 识别股票名称、板块、推荐等级与推荐理由
3. 补充股票代码
4. （可选）按市场过滤、按推荐等级排序、按板块分组
5. 输出 Markdown 表格

## 输出格式
```markdown
## [报告标题] 推荐股票清单

| 股票名称 | 股票代码 | 板块 | 推荐等级 | 推荐理由 |
|---------|---------|------|---------|---------|
| ... | ... | ... | ... | ... |

> 以上信息仅为报告内容整理，不构成投资建议。
```

## 使用示例
```
请分析 /path/to/report.pdf 中的股票

分析这份投研报告，剔除港股，按推荐等级排序

帮我提取这个PDF里推荐的股票，只要A股
```
