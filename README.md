# AlfonsSkills

## 简介
这是一个通用 skills 集合目录，用于存放可复用的任务技能。每个技能都包含 `SKILL.md`（规则与流程）、`README.md`（功能说明）以及必要的脚本与参考资料。

## 当前技能
| 名称 | 描述 | 路径 | 主要功能 |
| --- | --- | --- | --- |
| all-money-back-my-home | 分析投研 PDF 报告并输出结构化策略信息 | `all-money-back-my-home/` | 提取股票、板块、推荐等级与推荐理由，生成 Markdown 表格 |

## 使用方式
1. 进入技能目录
2. 阅读该技能的 `README.md` 与 `SKILL.md`
3. 按说明执行脚本或提供输入文件

## 目录结构
```
skills/
  README.md
  all-money-back-my-home/
    SKILL.md
    README.md
    LICENSE
    references/
    scripts/
```
